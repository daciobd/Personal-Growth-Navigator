Excelente material. São 8 abordagens distintas do "eu futuro" — cada uma com uma linguagem, pergunta-chave e objetivo diferentes. A forma mais elegante de instruir a IA a usá-las alternadamente é por **seleção automática baseada no perfil Big Five + adjetivos escolhidos**. Veja como implementar:

**Estratégia 1 — Seleção baseada no perfil (mais precisa)**

Cada abordagem do "eu futuro" tem uma indicação clínica natural:

- Alta Abertura + adjetivos criativos → **Narrativa** (re-autoria da história)
- Alto Neuroticismo + autocrítica → **CFT** (eu compassivo ideal)
- Baixa Conscienciosidade → **TFS** (pergunta do milagre + pequenos passos)
- Padrões rígidos detectados → **Terapia do Esquema** (adulto saudável)
- Alta Conscienciosidade + valores → **ACT** (direções de vida)
- Perfil equilibrado → **TCC** (visualização + custo-benefício)
- Alta Extroversão + relações → **Gestalt** (aqui-e-agora + cadeira vazia)

**Estratégia 2 — Rotação semanal (mais simples de implementar)**

A cada sessão de reavaliação ou nova jornada, a IA usa uma abordagem diferente. Isso expõe o usuário a múltiplas linguagens ao longo do tempo.

Aqui está o bloco de código pronto para adicionar ao `routes/plan.ts`:

```typescript
// artifacts/api-server/src/routes/plan.ts
// Adicionar ANTES do prompt principal

// Função de seleção automática por perfil Big Five
function selectApproach(
  big5?: { dims: Record<string, number> },
  futureAdjectives?: string[],
  sessionCount = 1
): FutureApproach {
  if (!big5) {
    // Sem Big Five: rotação pela sessão
    return FUTURE_SELF_APPROACHES[(sessionCount - 1) % FUTURE_SELF_APPROACHES.length];
  }

  const { N, E, O, A, C } = big5.dims as Record<string, number>;

  // Alta autocrítica/vergonha → CFT
  if (N >= 70) return FUTURE_SELF_APPROACHES.find(a => a.key === "cft")!;

  // Baixa Conscienciosidade → TFS (foco em pequenos passos concretos)
  if (C < 35) return FUTURE_SELF_APPROACHES.find(a => a.key === "tfs")!;

  // Alta Abertura → Narrativa (receptivo a metáforas e re-autoria)
  if (O >= 70) return FUTURE_SELF_APPROACHES.find(a => a.key === "narrativa")!;

  // Alta Amabilidade + baixo N → ACT (orientado a valores)
  if (A >= 65 && N < 50) return FUTURE_SELF_APPROACHES.find(a => a.key === "act")!;

  // Alta Conscienciosidade → Esquema (padrões profundos de desempenho)
  if (C >= 70) return FUTURE_SELF_APPROACHES.find(a => a.key === "esquema")!;

  // Alta Extroversão → Gestalt (presença e contato)
  if (E >= 65) return FUTURE_SELF_APPROACHES.find(a => a.key === "gestalt")!;

  // Padrão → TCC
  return FUTURE_SELF_APPROACHES.find(a => a.key === "tcc")!;
}
```

Depois, no prompt do Claude, adicione:

```typescript
// Dentro da função que monta o prompt em plan.ts:
const approach = selectApproach(big5Scores, futureAdjectives, assessmentNumber);

// Adicionar ao prompt:
`=== ABORDAGEM DO EU FUTURO ===
${approach.instruction}

Pergunta-chave para esta sessão: "${approach.question}"

Use essa perspectiva para fundamentar a síntese e a frase de intenção.
As práticas devem ENCARNAR essa abordagem específica — não apenas mencioná-la.`
```

E para retornar a abordagem usada ao frontend (para exibir ao usuário):

```typescript
res.json({
  success: true,
  plan: planData,
  approach: { name: approach.name, question: approach.question },
});
```

Isso faz com que o usuário veja, na tela do plano, algo como "Nesta jornada estamos usando a perspectiva da **Terapia Narrativa** — *Em quais momentos recentes você já agiu como a pessoa que quer se tornar?*" — o que tem valor educativo e torna cada reavaliação mensal genuinamente diferente das anteriores.
